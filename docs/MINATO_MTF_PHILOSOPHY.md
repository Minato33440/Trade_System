## 第1章 基盤理論 — ダウ構造

### 1-1. ダウ理論の3つの基本原則

```
原則①: 平均は全ての事象を織り込む
  → 価格にはファンダメンタルズを含む全ての情報が反映される
  → テクニカル分析が有効である根拠

原則②: トレンドは3段階からなる
  → 推進波（エリオット的には5波構成）+ 調整波（ABC）
  → 第3波が最も伸びる = エントリーの優位性はここが最も高い

原則③: トレンドは明確な転換シグナルが出るまで継続する
  → ダウの高値安値切り上げ/切り下げが崩れるまでトレンド継続
  → 「崩れ」の定義 = 戻り高値/押し安値の実体ブレイク
  → つまり初動だろうと9波目だろうと、崩れるまではトレンド方向にエントリーする
```

### 1-2. フラクタル構造

```
日足の1波
  ├── 4時間足の推進5波（1→2→3→4→5）
  │     ├── 1時間足のさらに細かい5波+ABC
  │     │     └── 15分足の...
  │     │           └── 5分足の...
  │     └── 4時間足のABC調整（A→B→C）
  └── 日足の2波 = 4時間足の別の推進5波（下方向）

全ての時間足が同じダウ構造を持つ。
上位足の1波の中に下位足の5波+ABCが内包される。
```

### 1-3. 裁量判断の3要素

```
要素①: 方向と勢い
  → 上位足トレンド方向のエリオット何波目か？
  → 3波が最も優位性高い（注文の集中が発生しやすい）
  → ただしこれはロット調整の材料であり、エントリー可否の判定ではない

要素②: 注文の集中
  → 上位足節目・トレンドライン・ネックライン
  → Fibonacci（特に61.8%圏内）
  → RSI・キリ番（150.000等）
  → これらが重なる「コンフルエンス」が強い支持/抵抗帯

要素③: ポジションの総数
  → 注文集中ポイントでの売買ネット総数の推定
  → 出来高（ボリューム）・ボラティリティの変化
  → プライスアクション（包み足・ピンバー等）による勢い判定
  → 大口によるストップ狩り（出来高急増＋下長髭）によるダマシを見抜く
    例: 4H押し目水準でSL狩り急落→急反転のリバースパターン
    → 出来高急増シグナルが検出できれば逆にエントリーの好機となる
```

## 第2章 MTF分析 — 4時間トレンドフォロー＋下位足同時3波

### 2-1. 主軸と補助の関係

```
★★★ 最重要な前提:
  主役は4時間足のトレンドフォロー。
  4Hダウが継続している限り、何波目であろうとエントリーする。
  （原則③: 明確な転換シグナルが出るまでトレンドは継続する）

  日足は「補助的なスコア要素」。
  → 日足がトレンド方向なら多少スコアが高くなる
  → 日足が逆張り方向なら多少スコアが低くなる
  → ただしエントリー可否を決定する要素ではない

  4Hの波カウント（3波/5波）はロット調整スコアに反映する。
  → 3波・5波は優位性が高い → ロット加算
  → 1波・7波以降は優位性がやや低い → ロット据え置きまたは減算
```

### 2-2. 各時間足の役割（トップダウン）
```
日足    → 補助スコア（トレンド方向の一致/不一致で加減点）
4時間足 → 主軸（ダウ切り上げ継続中の全ての押し目でエントリー判定）
1時間足 → 押し目の内部構造を確認（2番底の検出）
15分足  → エントリー水準の特定（ネックライン越え）
5分足   → エントリー確定足の判定 + 2波目以降の決済管理(注意：初動は15分ダウ崩れ決済）
```

### 2-3. 分析プロセス（裁量としての手順）

```
STEP ①: 日足の環境認識（補助）
  → 上昇トレンド / 下落トレンド / レンジのいずれかを判定
  → トレンド方向なら優位性スコア加点
  → 逆張りなら優位性スコア減点
  → ただしエントリー可否は4Hが決める

STEP ②: 4時間足の特定（主軸）
  → ダウ高値安値切り上げ確認（= 上昇トレンド継続）
  → 押し目 = 直近の4H SwingLow
  → 戻り高値（ネック） = 直近の4H SwingHigh（= neck_4h）
  → ★ ダウが継続している限り、何波目でもエントリー対象
  → 3波/5波はロット調整でスコア加算

STEP ③: 4時間押し目水準での3要素判定
  → 方向と勢い: 4時間足の波数＋日足方向一致を確認（スコア要素）
  → 注文の集中: Fib 61.8%圏内か / ネックとの距離は
  → ポジション総数: 出来高の変化 / 下ヒゲの出現等
  → 優位性スコアを統合

STEP ④: 1時間足の特定
  → 4時間押し目と同期する1時間SwingLow（2番底目）を検出
  → 安値切り上げ（ASCENDING）が理想形
  → ダブルボトム（DB）/ 逆三尊（IHS）も有効

STEP ⑤: 15分足の特定
  → 1時間2番底目の内部構造として15分の2番底を確認
  → 15分の戻り高値（ネックライン）越え = エントリー水準の特定
  → 5分足でネック越え実体確定 → エントリー執行

STEP ⑥: ボラティリティ係数（将来実装）
  → ゴールド100日平均と20日平均で算定
  → ボス個人のExcel関数表に基づく具体的な計算方式あり
  → 導入時に専用セッションで設計する

STEP ⑦: ロット調整（将来実装）
  → ボラティリティ係数 × 優位性スコア（日足方向 + 4H波カウント）
  → 優位性高+低ボラ = 最大ロット
  → 優位性低 or 高ボラ = 縮小ロット
```

### 2-4. 押し目パターンの4分類

```
①押し目買いエリアの安値切り上げ → ASCENDING（最も検出頻度が高い）
②押し目買いエリアの押し目買い   → DB（ダブルボトム）
③安値切り上げの安値切り上げ     → ASCENDING（連続型）
④安値切り上げの押し目買い       → IHS（逆三尊に近い構造）
```

### 2-5. 「トレンド転換した次の押し目を狙う」

```
核心的な概念（最も優位性が高い局面）:
  日足が上昇トレンド中に、4時間足がトレンド転換（高値安値切り上げ開始）
  → その4時間足の最初の押し目 = 第3波の起点
  → ここが最も伸びる場所

  ただし:
    4H第3波が「最も優位性が高い」ことは、
    「第3波以外でエントリーしない」ことを意味しない。

    ダウが継続している限り、全ての押し目がエントリー対象。
    第3波か否かはロット調整に反映される要素。
```

## 第3章 決済ルール

```
段階①: 15M初動1波 → 15Mダウ崩れで全量損切/利確
  → まだ15Mのスイング構造が確定する前の防御的決済
  → 広めのSLでノイズを吸収する局面

段階②: 15M 2波目以降 → 5分ダウ崩れで全量決済
  → 15Mトレンド転換が確定し、5Mトレンドフォロー中の決済
  → neck_4h未到達区間

段階③: 4時間ネック到達 → 半値決済 + 残りストップを建値移動
  → neck_4h（=4H戻り高値）に到達 = 利確ターゲット到達
  → ノーリスク化して残りを走らせる

段階④: 4時間ネック越え → 15分ダウ崩れで残り全量決済
  → 4H構造の新しいレジスタンスを超えた = 大きな推進中
  → 15Mダウが崩れるまでホールド
```
