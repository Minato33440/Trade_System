## 作業① swing_detector.py — フォールバック修正

### 修正対象: get_nearest_swing_high() / get_nearest_swing_low()

現在の実装（問題のある挙動）:
  Swing High/Low が lookback 内に検出できなかった場合
  → lookback 内の max() / min() をフォールバック値として返す
  → fib_range が常に大きな値になり MIN_4H_SWING_PIPS ガードが機能しない

修正後の仕様:
  Swing High/Low が検出できなかった場合 → None を返す
  検出できた場合 → 従来通り float 値を返す
```python
def get_nearest_swing_high(high: pd.Series,
                            n: int = 2,
                            lookback: int = 30) -> float | None:
    """
    直近の Swing High を返す。
    検出できない場合は None を返す（フォールバック値は返さない）。
    """
    window = high.iloc[-lookback:]
    mask   = detect_swing_highs(window, n=n)
    sh     = window[mask]

    if len(sh) == 0:
        return None   # ← 変更点: フォールバックなし

    return float(sh.iloc[-1])


def get_nearest_swing_low(low: pd.Series,
                           n: int = 2,
                           lookback: int = 30) -> float | None:
    """
    直近の Swing Low を返す。
    検出できない場合は None を返す（フォールバック値は返さない）。
    """
    window = low.iloc[-lookback:]
    mask   = detect_swing_lows(window, n=n)
    sl     = window[mask]

    if len(sl) == 0:
        return None   # ← 変更点: フォールバックなし

    return float(sl.iloc[-1])
```

重要: get_direction_4h() 等、他の関数が
get_nearest_swing_high/low() の戻り値を使っている箇所が
あれば全て None チェックを追加すること。
None のまま float 演算に渡すと TypeError になる。

### plot_swing_check() への追加

引数に sh_4h / sl_4h を追加し、Fib ラインを描画する。
```python
def plot_swing_check(df_5m        : pd.DataFrame,
                     sh_4h        : float | None,   # ← 追加
                     sl_4h        : float | None,   # ← 追加
                     direction_4h : str,
                     swing_highs  : pd.Series,
                     swing_lows   : pd.Series,
                     none_mask    : pd.Series,
                     save_path    : str) -> None:
    """
    Phase 1 + Fib ライン追加版。

    sh_4h / sl_4h が None の場合は Fib ラインを描画しない。
    """

    # --- Fib 計算 ---
    if sh_4h is not None and sl_4h is not None:
        fib_618 = sh_4h - (sh_4h - sl_4h) * 0.618
        fib_50  = sh_4h - (sh_4h - sl_4h) * 0.50

        # Fib 61.8% ライン（紫 #9B59B6 / 破線）
        ax.axhline(y=fib_618,
                   color='#9B59B6',
                   linewidth=1.0,
                   linestyle='--',
                   label=f'Fib 61.8%  {fib_618:.3f}',
                   alpha=0.8)

        # Fib 50% ライン（紫 #9B59B6 / 点線）
        ax.axhline(y=fib_50,
                   color='#9B59B6',
                   linewidth=1.0,
                   linestyle=':',
                   label=f'Fib 50%    {fib_50:.3f}',
                   alpha=0.8)
```

凡例に Fib 値（実数）を表示すること（上記 label= の形式）。
sh_4h / sl_4h が None の場合は Fib ラインなしで通常描画すること。

## 作業① swing_detector.py — 2関数追加

### 1-1. get_nearest_swing_high_1h()
```python
def get_nearest_swing_high_1h(high_1h: pd.Series,
                               n: int = 2,
                               lookback: int = 20) -> float | None:
    """
    1H足データから直近の Swing High を返す。
    neck_4h（押し目ゾーンの上限）として使用。

    検出できない場合は None を返す。

    引数:
      high_1h  : 1H足の High 系列
      n        : 前後確認本数（デフォルト2）
      lookback : 検索範囲（デフォルト20本 = 約20時間）
    """
    window = high_1h.iloc[-lookback:]
    mask   = detect_swing_highs(window, n=n)
    sh     = window[mask]

    if len(sh) == 0:
        return None

    return float(sh.iloc[-1])
```

### 1-2. get_nearest_swing_low_15m()
```python
def get_nearest_swing_low_15m(low_15m: pd.Series,
                               n: int = 3,
                               lookback: int = 20) -> float | None:
    """
    15M足データから直近の Swing Low を返す。
    Support_1h（押し目ゾーンの下限）として使用。

    check_15m_range_low() の sl_min とは独立して取得する。
    sl_min は構造検出用（頭の最深値）。
    この関数は「現在のサポートライン」取得用。

    検出できない場合は None を返す。

    引数:
      low_15m  : 15M足の Low 系列
      n        : 前後確認本数（デフォルト3、#014確定値）
      lookback : 検索範囲（デフォルト20本 = 約5時間）
    """
    window = low_15m.iloc[-lookback:]
    mask   = detect_swing_lows(window, n=n)
    sl     = window[mask]

    if len(sl) == 0:
        return None

    return float(sl.iloc[-1])
```

### 追加: get_nearest_swing_low_1h()
```python
def get_nearest_swing_low_1h(low_1h: pd.Series,
                              n: int = 2,
                              lookback: int = 240) -> float | None:
    """
    1H足データから直近の Swing Low を返す。
    Support_1h（押し目ゾーンの下限）として使用。

    lookback=240 の根拠:
      4H波形成に約1週間（チャート実測値）
      安全マージンを含め10日分を確保
      10日 × 24本/日（1H足）= 240本

    これは get_nearest_swing_low_15m（lookback=20）の
    正式な上位足版として実装する。

    検出できない場合は None を返す。
    """
    window = low_1h.iloc[-lookback:]
    mask   = detect_swing_lows(window, n=n)
    sl     = window[mask]

    if len(sl) == 0:
        return None

    return float(sl.iloc[-1])
```
