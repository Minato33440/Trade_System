MTF v2 バックテストテンプレート（2026/03/13現在版）
目的

直近2年Polygon無料データ（5m/15m/1h/4h/日足）を使用
4H優位性（3波構造）ベースの短期MTFルールでロング/ショート両方を集計
日足トレンドルール（EMA20/50 GC or ADX>25）を「オプション」としてON/OFF比較
最大DD（一気に吐き出すリスク）を最優先に把握
手動裁量支援ツールとして「実損益・期待値・PF」を明確に数値化

テスト条件（デフォルト）

通貨ペア：USDJPY
期間：直近2年（2024-03-13 〜 2026-03-12）
Lotサイズ：1Lot（10万通貨）
リスク：1トレードあたり資金の0.5%
手数料：0.0005（0.05%）
スリッページ：0.0002
時間帯フィルタ：東京(9-11)、ロンドン(17-19)、NY(22:30-翌1:00)、all

出力項目（セッション別＋全体）

総トレード数（ロング/ショート別も）
4H優位性シグナル数（エントリー前フィルタ通過数）
Lot数
平均利確Pips
平均損切Pips
2年損益合計（円換算、10万通貨基準）
期待値（Expectancy = (勝率×平均勝ちPips) - (負率×平均負けPips)）
Profit Factor
最大DD（%）
勝率（4H優位性のみ）
勝率（日足ルール追加時）
ロング/ショート別勝率・損益内訳

比較パターン

4H優位性のみ（日足ルールOFF） → シグナル数増加・勝率低下の可能性
日足ルール追加（EMA20/50 GC or ADX>25） → シグナル数減少・質向上の可能性

リスク管理の重点指標

最大DD：10%超えなら即見直し（一気に吐き出すリスクをボスが最重視）
期待値：+5pips以上で「裁量で狙う価値あり」
PF：1.5以上で「長期的にプラス」
ロング/ショート別：USDJPYの特性（ショート強い時期多め）を反映

実装イメージ（backtest.py内）

run_usdjpy_mtf_v2_advanced()関数で上記全てを計算
mtf_minato_short_v2()をuse_daily=True/Falseで2回呼び出し
VectorBTまたは自前計算でロング/ショート別損益・DD・PFを算出
結果をMarkdownテーブルで出力（セッション別＋比較表）

ボス、これでテンプレートの全体像が一発でわかるはず。
このmdを基に「期待値+5pips以上なら緑色表示」「MaxDD10%超で赤警告」みたいな視覚化も後で可能。
