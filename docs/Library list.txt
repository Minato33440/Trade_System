# REX_Trade_System インストール済みライブラリ一覧（2026/03/11 時点）

REX_Trade_Systemの仮想環境でインストール済みの主なライブラリと、その役割・使いどころをまとめました。  
基本的にGMトレード戦略構築とローカルシステム運用に特化しています。

## コア分析・データ処理系

| ライブラリ          | 主な役割                                                                 | 具体的な使いどころ例                                                                 | 優先度 |
|---------------------|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------|--------|
| pandas             | 表形式データの読み書き・加工・分析の標準ツール                           | 取引履歴CSV読み込み、yfinanceデータ整理、バックテスト結果集計                       | ★★★★★ |
| numpy              | 高速数値計算・配列操作                                                   | テクニカル指標（MA, RSI, ボラ）計算、乱数によるシンクロニシティシミュレーション   | ★★★★★ |
| yfinance           | Yahoo Financeから無料でヒストリカル/リアルタイム市場データ取得           | USD/JPY, EUR/USD, 金, 原油, 株価指数などのGM関連データ取得                         | ★★★★☆ |
| requests           | HTTPリクエスト（外部API呼び出し）                                       | xAI Grok API, Polygon API, Coingecko API など外部データ取得                        | ★★★★★ |
| feedparser         | RSSフィード解析                                                         | `--news` のRSS取得・パース（見出し抽出、リンク追跡）                                | ★★★★☆ |
| beautifulsoup4     | HTML解析（本文抽出/ノイズ除去）                                          | `--news` のリンク先本文要約のためのテキスト抽出                                     | ★★★☆☆ |

## UI・可視化系

| ライブラリ          | 主な役割                                                                 | 具体的な使いどころ例                                                                 | 優先度 |
|---------------------|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------|--------|
| streamlit          | ブラウザで動く簡単Webアプリ（ダッシュボード）                           | dashboard.py でチャート表示 + 最新シグナル + Rexチャット埋め込み                   | ★★★★★ |
| matplotlib         | 静的グラフ描画（チャート・バックテスト結果可視化）                       | 価格チャート、累積リターン曲線、シグナル発生ポイントのプロット                     | ★★★★☆ |

## 環境・セキュリティ系

| ライブラリ          | 主な役割                                                                 | 具体的な使いどころ例                                                                 | 優先度 |
|---------------------|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------|--------|
| python-dotenv      | .envファイルからAPIキーなどの環境変数を安全に読み込み                   | XAI_API_KEY, POLYGON_API_KEY の管理（コードに直書き禁止）                           | ★★★★★ |

## その他（依存で自動インストールされたもの）

- tqdm：進捗バー表示（データ取得時などに便利）
- certifi / charset-normalizer / urllib3：HTTP通信の安定性・文字コード処理
- ほか多数の依存パッケージ（基本的に裏方で支えてくれる）

## 今後の拡張候補（まだ入れてないけどおすすめ）

- polygon：本格的な高頻度・リアルタイムデータ取得（キー持ってるなら即導入）
- plotly：インタラクティブなチャート（Streamlitと相性抜群）
- ta-lib または pandas_ta：テクニカル指標の計算をさらに楽に
- backtrader / vectorbt：本格バックテストエンジン

## 注意点
- .gitignore に `.env` と `__pycache__` 系は必ず入れてセキュリティ確保
- requirements.txt は「運用に必要な最小セット」を正本として管理（`pip freeze` による全固定は原則しない）
- Polygon API キー持ってるなら、yfinance から段階的に移行すると精度が上がるよ
